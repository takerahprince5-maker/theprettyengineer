Description

This project is a Python-based right triangle trigonometry calculator that uses the math library to solve for missing side lengths and angles. I created functions to calculate the adjacent side given the hypotenuse and angle, the opposite side given the hypotenuse and angle, the angle given the hypotenuse and opposite side, and the angle given both the adjacent and opposite sides. The program highlights my ability to combine programming logic with mathematical formulas to solve real-world problems.

Skills Learned

  • Applying trigonometric functions (sine, cosine, tangent) and inverse trigonometric functions in Python
  • Using the math library effectively to perform calculations
  • Designing reusable functions for problem-solving
  • Strengthening understanding of right triangle relationships
  • Translating mathematical concepts into code implementations
