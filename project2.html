Description

This project implements the Hill cipher, a classical encryption algorithm that uses linear algebra to secure alphabetic text. The program reads a key matrix (between 2×2 and 9×9) from one file and plaintext from another file, both provided as command-line parameters. After parsing the inputs, it prints the key matrix and the plaintext, performs the encryption using matrix multiplication and modular arithmetic, and then outputs the resulting ciphertext to the console. The implementation demonstrates how cryptographic techniques can be translated into code while adhering to strict input/output and formatting requirements.

Skills Learned

  • Applying linear algebra concepts (matrix multiplication and modular arithmetic) to cryptography
  • Implementing the Hill cipher encryption algorithm programmatically
  • Parsing and processing input from external files in Java
  • Handling command-line parameters to control program execution
  • Designing structured output for readability and validation
  • Strengthening understanding of classical cryptography and its role in modern security practices
